glue genes leverages the power of glue to help you understand your genomics data

Linked Data and Visualizations

glue genes builds on the glue library -- users can specify logical connections between multiple datasets and easily create different views of their data which will all be connected so that selections in any graph propagate to all others.

Customized for Genomics

glue genes includes data loaders, custom viewers, and analysis plug-ins to give you what you need for exploring genomics and transcriptomics data. See the documentation page for more.

Open Source and Extensible

glue genes is an open source Python library with a robust plug-in architecture, making it easy for users to add their own custom viewers or analysis routines from their favorite external packages. Consulting services are also available.
